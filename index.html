🔥 Configura Firebase

Para que todos vean los resultados en tiempo real necesitas conectar Firebase (gratis). Sigue estos 3 pasos:

PASO 1 — Crear proyecto

Ve a console.firebase.google.com → Add project → ponle un nombre → Continue (sin Analytics) → Create project

PASO 2 — Realtime Database

En el menú: Build → Realtime Database → Create database → Start in test mode → Enable

PASO 3 — Obtener configuración

En Project settings (⚙️) → General → Your apps → </> Web → registra app → copia los valores de firebaseConfig y pégalos abajo:

La configuración se guarda en tu navegador. Solo lo configuras una vez.
